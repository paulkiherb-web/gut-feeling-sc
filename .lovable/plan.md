

# GreenRed AI — Smart Food Scanner MVP

## Обзор
Полноценный MVP "умного сканера еды", который персонализирует рекомендации на основе биологического профиля пользователя. Дизайн в стиле "Clinical Luxury" — минималистичный, премиальный, как Apple Health.

---

## Фаза 1: Дизайн-система и основа

Настройка цветовой палитры (Emerald/Amber/Rose), шрифт Inter, скругления 24px, glassmorphism-эффекты для карточек. Тёмная/светлая тема с адаптивным переключением.

---

## Фаза 2: Onboarding (3 шага с анимацией)

- **Шаг 1 — "Биометрия"**: Переключатель пола (M/F/Other), слайдер возраста 1–100
- **Шаг 2 — "Текущее состояние"**: Визуальные карточки с иконками — Здоров, Беременность, После операции (+ поле "дней после операции"), Хронические заболевания, Атлет
- **Шаг 3 — "Цель"**: Чипсы выбора — Похудение, Максимум энергии, Восстановление, Лучший сон
- **Финал**: Анимация "Создаём ваш биологический двойник..." с пульсирующим лого

Данные сохраняются в профиль пользователя.

---

## Фаза 3: Paywall-экран

Glassmorphism дизайн с размытым фоном еды. Список преимуществ: безлимитные сканирования, AI-доктор контекст, анализ лабораторных (скоро). Два тарифа: $9.99/мес и $59.99/год ("Best Value"). Кнопка "Попробовать 7 дней бесплатно". Логика: 3 бесплатных скана в день для free-пользователей.

---

## Фаза 4: Главный экран — Сканер

- Полноэкранный интерфейс камеры (placeholder-изображение + кнопка загрузки фото)
- **Context Switcher** вверху — pill-кнопки для быстрой смены состояния ("По умолчанию" → "После операции")
- Футуристичная рамка сканирования в центре со светящимися углами
- Большая кнопка "Scan Food"
- Мини-лента последних 3 сканирований внизу с цветовыми индикаторами (🟢🟡🔴)

---

## Фаза 5: Drawer с вердиктом

Выезжающая снизу панель после сканирования:
- Большой цветной индикатор (зелёный круг / жёлтый треугольник / красный крест)
- Название блюда (распознанное AI)
- AI-объяснение: почему это подходит/не подходит конкретно этому пользователю
- Альтернативное предложение при Yellow/Red результате

---

## Фаза 6: Health Profile

Дашборд с текущим профилем пользователя, историей сканирований, статистикой (сколько Green/Yellow/Red за неделю).

---

## Фаза 7: Backend (Supabase + OpenAI)

- **База данных**: Таблицы `user_profiles` (возраст, пол, состояние, цель, дни после операции) и `scans` (фото, название, вердикт, анализ, таймстамп)
- **Edge Function `analyze-food`**: Принимает base64 изображение + профиль пользователя → отправляет в GPT-4o Vision → возвращает JSON с вердиктом, объяснением и альтернативой
- **Секрет**: OpenAI API ключ через Supabase Secrets

---

## Навигация

Onboarding → Paywall → Scanner (главный экран) → Analysis Drawer → Health Profile (через боковое меню или таб)
